Detailed Description

Print too many messages.

Since
version 1.0.0

Verbose mode

The verbose mode status can be inspected using verbose_mode_state(). It can either be enabled or disabled using verbose_mode_enable() or verbose_mode_disable().

Usage

When the verbose mode is enabled, the following functions behave as their *printf(3) counterparts:

verbose printing function *printf(3) function
verbosef() printf(3)
fverbosef() fprintf(3)
sverbosef() sprintf(3)
snverbosef() snprintf(3)

On the other hand, if the debugging mode is disabled, these functions do nothing except returning 0.
