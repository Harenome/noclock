Detailed Description

Arithmetic or boolean expression structs.

Since
version 1.0.0

This module declares the expression_type enum and the expression struct.

An expression can be:

  • an identifier (its type would be EXPR_ID).
  • a number (its type would be EXPR_NUMBER).
  • a boolean value (its type would be either EXPR_TRUE or EXPR_FALSE).
  • an unary operation on another expression (its type would be either EXPR_NEG or EXPR_NOT).
  • a binary operation on two expressions (its type would be any expression_type apart from EXPR_UNKNOWN and the aforementioned types).

For further information on:

  • expression constructors or destructors, see Constructors / Destructors.
  • expression getters, see Getters.
  • expression setters, see Setters.
  • expression operations, see Operations.
  • expression input/output, see Input / Output.

To use lists of expression, see Lists of expressions.
