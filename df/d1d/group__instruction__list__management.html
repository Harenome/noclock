Detailed Description

Create or destroy instruction_list lists.

An instruction_list node can be created using instruction_list_alloc().

An instruction_list list must be destroyed using instruction_list_free() or softly destroyed using instruction_list_soft_free().

Softly freeing an instruction_list list using instruction_list_soft_free() destroys the container list without destroying the inner instruction elements whereas the "standard" freeing with instruction_list_free() destroys the list and its elements.
