Detailed Description

Set properties of expressions.

Since
version 1.0.0

These functions allow the user to set the properties of an expression. Please set coherent properties. If you are not confident a given set of properties would be coherent, please fallback to the functions from Constructors / Destructors and Operations. The functions from these modules automatically set the appropriate properties.
