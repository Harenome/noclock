Detailed Description

Lists of expression structs.

Since
version 1.0.0

This module declares the expression_list struct which is used to handle lists of expression structs.

For further information on:

  • expression_list constructors or destructors, see Constructors / Destructors.
  • expression_list getters, see Getters
  • expression_list operations, see Operations
  • expression_list input/output, see Input / Output

Lists of expression structs are linked lists. An expression_list struct is a node which refers to an expression and to the next expression_list node.

Given an expression_list named list, the recommended way to iterate on its elements is as follows:

for (expression_list * node = list; node != NULL; node = node->next)
{
expression * expr = node->element;
// Use expr.
}
