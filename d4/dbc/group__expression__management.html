Detailed Description

Create or destroy expressions.

Since
version 1.0.0

Blank expressions can be created with expression_alloc(). Any expression created via expression_alloc() can be destroyed with expression_free(). An expression can be copied with expression_copy().

It is also possible to directly create constant expressions from:

  • a boolean value: expression_from_boolean()
  • an identifier: expression_from_identifier()
  • a number: expression_from_number()
Warning
It is strongly advised not to use expression_init() and expression_clean() unless you know what you are doing!
