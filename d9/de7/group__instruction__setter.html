Detailed Description

Get properties of instructions.

Since
version 1.0.0

These functions allow the user to set the properties of an instruction. Please set coherent properties. If you are not confident a given set of properties would be coherent, please fallback to the functions from Constructors / Destructors to directly build a correct instruction.
