Append an expression to an expression list.

Since
version 1.0.0
Parameters
listList where the expression shall be appended.
expressionExpression to append.
Returns
The resulting list.
Note
The pointer returned by this function may be equal to the input list.
Warning
This function may modify the input list but do not assume the pointer to the input list may still be safely used and please use the returned pointer.
The correct way to use this function is to store the return value, as follows:
list = expression_list_append (list, expr);
Whereas discarding the return value as follows is an incorrect use of this function: as follows:
expression_list_append (list, expr);
